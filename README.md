# Agentic AI Price Prediction Capstone 🕵️‍♂️💸

A multi-agent AI system that autonomously scans deal feeds, estimates a product's true value using an ensemble of pricing models (a fine-tuned LLM, a RAG-based frontier model, and a neural network), and sends a push notification when it finds a genuine bargain.
